# kkongchii 포트폴리오 홈페이지

개인 포트폴리오 홈페이지입니다. 빌드 도구 없이 순수 HTML/CSS/JS로 구현했습니다.

🔗 **배포 URL**: https://myhomepage-xi.vercel.app/

---

## 구성 섹션

| 섹션 | 내용 |
|------|------|
| Hero | 이름, 역할(백엔드 개발자 & 데이터 엔지니어), 소셜 링크 |
| 기술 스택 | Java, Python, JS, Spring Boot, React, MySQL, DB2, Oracle 등 |
| 경력 | 현대HDS (AI 포탈, 데이터 파이프라인, 데이터마트, BI) |
| 프로젝트 | 현업 2개 + 개인 GitHub 프로젝트 2개 |
| 연락처 | 이메일, GitHub, Velog |

## 기능

- 다크/라이트 모드 토글 (선택값 localStorage 유지)
- 반응형 디자인 (모바일 햄버거 메뉴 포함)
- 스크롤 페이드인 애니메이션
- 네비게이션 바 스크롤 하이라이트

## 로컬 실행

별도 빌드 없이 `index.html`을 브라우저에서 직접 열면 됩니다.
